# Qbot Pro V5.20

Qbot Pro 是面向机器人结构概念设计的单文件快速估算平台，用于尽快收敛方案、形成选型门槛和暴露缺失数据，不替代专业标准计算、动力学仿真或实机验证。

- 在线使用：https://yueqiang-robotics.github.io/Qbot-calculator/
- 离线使用：下载本仓库的 `index.html` 后直接用浏览器打开
- 作者：YUEQ（岳强）
